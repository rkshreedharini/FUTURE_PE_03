## Overview
This repository contains my submission for Task 3 of the Future Interns internship program. The objective of this task was to design and document chatbot personas using prompt engineering techniques and deploy one working chatbot using a no-code platform.

---

## Project Description
The project focuses on creating distinct chatbot personalities with defined roles, tones, and behaviors. Each chatbot persona was designed to handle real-world use cases with consistent communication and user-centered conversation flow.

---

## Chatbot Personas
The following chatbot personas were designed and documented:

1. AI Mental Health Companion  
   - Focuses on emotional support and stress relief through empathetic conversations.

2. AI Subject Tutor  
   - Provides clear explanations, learning assistance, and academic support.

3. AI Travel Guide  
   - Assists users with travel planning, destination suggestions, budgets, and packing tips.
